Context rot is real.

Early in a chat your agent one-shots the function. A few hundred messages later it re-introduces the bug you fixed an hour ago and confidently invents an API. You can't feel it coming — the meter can.

how it works

Boringly robust.

// reads the real number

Pulls the actual context-window % straight from Claude Code's status-line data. No guessing, no transcript scraping.

// zero dependencies

One small Node script. Nothing to install, nothing to break. Runs locally on every render — costs you zero tokens.
